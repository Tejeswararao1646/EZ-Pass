# EZ Pass 
EZ Pass is a simple web application for applying for a bus pass.  
It allows users to enter their details, upload documents, and complete a basic application process in an easy way.

---

## Features

- Login & Signup interface  
- Application form (Student / Employee / General)  
- Dynamic fields based on user type  
- File upload inputs:
  - Photo / Aadhaar Copy  
  - Student ID / Bonafide Certificate  
  - Address Proof  
- Form validation:
  - Username (letters & numbers only)  
  - Password (8+ characters with letter, number, symbol)  
  - Name (letters only)  
- Save & Resume functionality  
- QR code payment simulation  
- Clean and responsive UI  

---

## Technologies Used

- HTML  
- CSS  
- JavaScript
  
---

##  How to Run

1. Download or clone this repository  
2. Open `index.html` in any browser  

---

## Note

This is a frontend-only project:

- No backend integration  
- No database  
- File uploads are not stored  
- Payment is only simulated
